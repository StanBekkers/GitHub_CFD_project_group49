# CFD GPU cooling project

## Description
CFD project that simulates GPU cooling. 
Using waterflow through different fin.
Finding temperatures of both the walls. 
And temperatures, pressures and velocities of the cooling water. 
---

## Project Structure

- `scripts/` → main scripts to run
- `functions/` → reusable MATLAB functions
- `data/` → input data
- `results/` → generated results (plots, outputs)

---

## How to Run

1. Open MATLAB
2. Navigate to the project folder
3. Run: 
     ->put in certain fins designs as (csv)
     ->put in system parameters such as P_wall, V_water etc. 
     ->scripts/GPU_Cooler.m
